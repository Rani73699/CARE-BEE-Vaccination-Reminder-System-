# CARE-BEE-Vaccination-Reminder-System-
With the rapid development of economy and technology, people have simply not being  satisfied  with the level of medical treatment and care  but also began to pay more attention  towards  health care and  environment gradually. CARE BEE is the first step in treatment of  children, and also a crucial component in children's hospital functional block.Therefore, Children vaccination reminder is an application for tracking baby’s vaccination.To create a healthy, harmonious, and sustainable development of eco-child health care space by arrangement of CARE BEE  has become more significant for the care of new born lives and children’s vaccination as well. 



  #Software Requirements

Operating System : Windows XP or above
Framework : Django
Frontend : HTML5,CSS3,Javascript,Bootstrap
Middleware : Python
Backend : MySQL
Tools Used : Sublime Text Editor
